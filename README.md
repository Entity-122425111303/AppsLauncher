# AppsLauncher
### 一、应用介绍
顾名思义，这是一款应用启动器，用于启动应用，可以让你摆脱一次次寻找，在任何电脑上点击直达应用
### 二、应用功能
- 主要功能：通过此应用方便地打开任何其他应用
- 详细介绍：
  1. 应用中有增加按钮，可以增加目标应用
  2. 还有删除、编辑、打开按钮
  3. 列表下方有设置
  4. 完整操作（示例）：
     1. 点击增加按钮增加想打开的应用，可以点击浏览快速添加
     2. 在列表中点击应用名
     3. 点击打开按钮
     4. 下次打开此应用依然在列表中，方便直接打开
